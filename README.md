# NYC Old World Trade Center Cityscape

Interactive 3D cityscape of New York City (pre-2001),
featuring the original World Trade Center Twin Towers.

## Features
- 3D Twin Towers (North & South towers not perfectly aligned)
- Lower Manhattan skyline
- Day / night toggle
- Orbit camera controls

## Tech Stack
- React
- Three.js
- @react-three/fiber
- @react-three/drei
- Vite

## Install
npm install

## Run
npm run dev

## Notes
This project is a stylized representation.
Exact geospatial accuracy can be added using real coordinates and models.
